# Accretion–Dilution M&A Model

## 📌 Overview

This project is a financial model built to analyze whether an acquisition is **accretive or dilutive** to the acquirer’s Earnings Per Share (EPS).

It simulates a merger scenario by combining buyer and target financials, applying deal assumptions, and evaluating the post-transaction impact.

---

## 🎯 Objective

To determine how an acquisition affects shareholder value by analyzing EPS changes under different deal structures.

---

## 🧩 Key Features

* Buyer & Target financial analysis
* Deal assumptions (cash, debt, or stock financing)
* Pro forma financial statements
* EPS impact (Accretion/Dilution analysis)
* Scenario-based evaluation

---

## 📊 Key Insight

The model evaluates whether the transaction is **accretive (increases EPS)** or **dilutive (decreases EPS)** based on financing structure and earnings contribution of the target company.
